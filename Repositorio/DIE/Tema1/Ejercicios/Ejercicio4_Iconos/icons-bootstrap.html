A単adir iconos

Para insertar los iconos de Bootstrap en nuestra pagina web, necesitaremos el siguiente link a la hoja de estilos

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">

Para poder insertar un icono en nuestra pagina web tendremos que utilizar la etiqueta <i> con el atributo class al cual asignaremos "glyphicon glyphicon-eye-open" para indicar al icono

Si queremos utilizar un icono en concreto tendremos que ponerle dentro de le etiqueta class despues de la primera clase el icono que queremos utilizar, en este caso anterior seria glyphicon-eye-open
