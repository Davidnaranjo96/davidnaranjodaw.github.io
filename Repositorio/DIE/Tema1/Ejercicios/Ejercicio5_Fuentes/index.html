Como añadir Fuentes de Google

Para instalar una fuente de google utilizaremos como ejemplo la fuente Rowdies. Lo haremos paso a paso:

Selecionamos la fuente y la familia

Podemos instalarlo de 2 maneras, a traves de un link en el head o en el import

1-Link


2-Import

Instalacion con el link

Añadiremos a nuestro html entre la etiqueta el siguiente link

<link href="https://fonts.googleapis.com/css2?family=Rowdies:wght@300&display=swap" rel="stylesheet">

Instalacion con import

Esta es la forma más recomendada de hacerlo. Insertamos en primera línea donde tengamos nuestro CSS el código generado

@import url('https://fonts.googleapis.com/css2?family=Rowdies:wght@300&display=swap');

Para utilizarlo simplemente tendremos que aplicarlo en el css o en un archivo externo o con la etiqueta style
